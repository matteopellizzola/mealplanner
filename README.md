## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Progressive Web App (PWA) Support

This application includes a web manifest and icons to make it installable on mobile devices.

- **Android / Chrome**: Open the site, tap the browser menu and choose "Add to Home screen". The app will launch full-screen without browser UI.
- **iOS / Safari**: Visit the site, tap the share icon, and select "Add to Home Screen". Make sure the manifest and `apple-touch-icon` are referenced correctly.

Replace the placeholder icons in `public/icons/` with properly sized `192x192` and `512x512` PNG images for best results.
